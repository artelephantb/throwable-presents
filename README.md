## About
**Throwable Presents** is a *datapack* for *Minecraft Java Edition*. This datapack introduces new items, recipies, and content to Minecraft!

## Items and Blocks

### Present
**Presents** can be thrown a small distance; When the present lands, it creates a storage container ready for you to fill it!

> [!NOTE]
> Presents do not contain any contents when placed, the player has to fill them up.

![Present Recipe](/images/recipies/present.png)

### Candy Cane
Run fast with the **Candy Cane**, adding *500%* speed to the player when held.
![Candy Cane Recipe](/images/recipies/candy_cane.png)

### Gingerbread
A delicous treat. The **Gingerbread** has nutrition of *5*, and saturation of *2*.
![Gingerbread Recipe](/images/recipies/gingerbread.png)

## Powtensia Staff Support
*Throwable Presents* works with *Swrontiope*, including the *Powtensia Staff*. When using a **Present** in the offhand, and the Powtensia Staff in the main hand, the thrown present will create a pile of presents in the direction of the shot.
